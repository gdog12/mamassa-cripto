# 🚀 MaMassa Cripto

**MaMassa Cripto** is a comprehensive, secure, and modern desktop application designed for tracking cryptocurrency market trends, corporate holdings, and industry news in real-time. Built with performance and user experience in mind, it provides a clean, customizable interface for both casual observers and serious crypto enthusiasts.

---

## ✨ Features

- 🔒 **Secure Access:** First-run password creation (minimum 8 characters, alphanumeric) to keep your data private.
- 📈 **Live Market Trends:** Real-time data for the top 100 cryptocurrencies, complete with a dynamic bar chart showing the top 10 by 24h trading volume.
- 🏢 **Corporate Holdings Tracker:** A curated list of 100+ public companies, miners, and financial institutions with significant cryptocurrency exposure.
- 📰 **Scrolling News Ticker:** A highly visible, smooth-scrolling ticker tape at the top of the app displaying the latest crypto industry news. Click any headline to open the full article in your browser.
- 🎨 **5 Eye-Pleasing Themes:** Customize your experience with professionally designed color palettes:
  - Dark Purple (Default)
  - Soft Blue
  - Warm Orange
  - Gentle Pink
  - Clean Light
- 🔗 **Exchange Directory:** A quick-reference list of trusted, regulated cryptocurrency exchanges with direct links to their sign-up pages.
- 🔄 **Auto-Refresh:** Market data and news update automatically in the background without freezing the UI.

## 🛠️ Installation & Usage

### Option 1: Pre-Compiled Executable (Recommended for Windows)
1. Download the latest `MaMassa_Cripto.exe` from the [Releases](https://github.com/gdog12/mamassa-cripto/releases) page or GitHub Actions artifacts.
2. Double-click the `.exe` file to run. 
3. *Note: Windows SmartScreen may show a warning for newly compiled apps. Click "More info" -> "Run anyway".*

### Option 2: Run from Source (Windows / Linux)
1. Ensure you have **Python 3.10+** installed.
2. Clone this repository:
   ```bash
   git clone https://github.com/gdog12/mamassa-cripto.git
   cd mamassa-cripto
